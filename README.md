# Nomad Golf — Website

Website for Nomad Golf / Golf Gadget product.

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy to Vercel

```bash
npm install -g vercel
vercel
```

Or connect the GitHub repo to Vercel and it deploys automatically on every push.
